# 1️⃣1️⃣ Lyrics Screen (اختياري)

## الوصف

عرض كلمات الأغنية إذا كانت متوفرة، إما مضمّنة في الملف (ID3 Tags) أو من مصدر خارجي.

---

## مصادر الكلمات

| المصدر | الأولوية | الملاحظة |
|--------|---------|----------|
| ID3 Tag (USLT) | 🥇 أولاً | مضمّنة في ملف الـ MP3 |
| ملف `.lrc` بجانب الأغنية | 🥈 ثانياً | نفس الاسم ونفس المجلد |
| مصدر خارجي (مستقبلياً) | 🥉 ثالثاً | API خارجي |

## صيغة LRC

```lrc
[ti:اسم الأغنية]
[ar:الفنان]
[al:الألبوم]
[00:12.50] السطر الأول من الأغنية
[00:15.20] السطر الثاني من الأغنية
[00:18.00] السطر الثالث
```

---

## التنفيذ المبسط

```dart
// قراءة ملف LRC
List<LrcLine> parseLrc(String lrcContent) {
  final lines = <LrcLine>[];
  final regex = RegExp(r'\[(\d{2}):(\d{2})\.(\d{2})\](.+)');
  for (final match in regex.allMatches(lrcContent)) {
    final minutes = int.parse(match.group(1)!);
    final seconds = int.parse(match.group(2)!);
    final centiseconds = int.parse(match.group(3)!);
    final text = match.group(4)!.trim();
    final time = Duration(
      minutes: minutes,
      seconds: seconds,
      milliseconds: centiseconds * 10,
    );
    lines.add(LrcLine(time: time, text: text));
  }
  return lines;
}
```
