# 1️⃣2️⃣ Search Screen

## الوصف

بحث شامل وسريع في جميع المكتبة الموسيقية.

---

## التخطيط


---

## نطاق البحث

| النوع | الحقول المبحوثة |
|-------|----------------|
| أغاني | اسم الأغنية + الفنان + الألبوم |
| فنانين | اسم الفنان |
| ألبومات | اسم الألبوم + الفنان |
| Playlists | اسم الـ Playlist |

---

## المنطق

```dart
// Debounce للبحث الفوري
final _searchController = TextEditingController();
Timer? _debounce;

void _onSearchChanged(String query) {
  if (_debounce?.isActive ?? false) _debounce!.cancel();
  _debounce = Timer(const Duration(milliseconds: 300), () {
    ref.read(searchProvider.notifier).search(query);
  });
}
```

---

## تجربة المستخدم (UX)

- البحث يبدأ من **3 أحرف** فصاعداً
- نتائج فورية بـ debounce 300ms
- تسليط الخط (highlight) على الجزء المطابق من النص
- حفظ آخر 10 عمليات بحث محلياً
- زر `✕` لمسح البحث والعودة للحالة الفارغة
