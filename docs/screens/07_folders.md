## المنطق (Logic)

```dart
// استخراج المجلدات من قائمة الأغاني
Map<String, List<Song>> groupByFolder(List<Song> songs) {
  final Map<String, List<Song>> folders = {};
  for (final song in songs) {
    final folderPath = song.data.substring(0, song.data.lastIndexOf('/'));
    final folderName = folderPath.split('/').last;
    folders.putIfAbsent(folderName, () => []).add(song);
  }
  return folders;
}
```

## ملاحظات مهمة

- **تجاهل الملفات القصيرة**: بإمكانية ضبط حد أدنى للمدة (مثلاً 30 ثانية) من الإعدادات
- المجلدات تُحسب ديناميكياً من بيانات `on_audio_query` — لا تخزين منفصل
- رتّب المجلدات بعدد الأغاني تنازلياً بالافتراضي
