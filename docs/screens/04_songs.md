# 4️⃣ Songs Screen

## الوصف

تعرض جميع الأغاني الموجودة على الجهاز مع خيارات ترتيب وبحث.

---

```

├─────────────────────────────────┤
```

## خيارات الترتيب

| الترتيب | الوصف |
|---------|-------|
| الاسم (A-Z) | ترتيب أبجدي تصاعدي |
| الاسم (Z-A) | ترتيب أبجدي تنازلي |
| الفنان | تجميع حسب الفنان |
| الألبوم | تجميع حسب الألبوم |
| التاريخ (الأحدث) | حسب تاريخ الإضافة |
| التاريخ (الأقدم) | |
| المدة (الأطول) | |
| المدة (الأقصر) | |

---

## قائمة السياق (Long Press / ⋮)

عند الضغط المطول على أغنية أو النقر على `⋮`:

```
▶ تشغيل
⏭ تشغيل بعد الحالية
➕ إضافة للقائمة
❤ إضافة للمفضلة
💿 الانتقال للألبوم
🎤 الانتقال للفنان
📂 عرض في المجلد
ℹ تفاصيل الأغنية
```

---

## البحث

- البحث يعمل **فورياً** (debounce 300ms)
- يبحث في: اسم الأغنية + الفنان + الألبوم
- إذا كان النص فارغاً تظهر القائمة الكاملة

---
