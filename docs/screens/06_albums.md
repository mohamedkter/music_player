## ميزة Dynamic Color

عند فتح تفاصيل الألبوم:
1. استخراج اللون السائد من صورة الغلاف باستخدام `palette_generator`
2. تطبيق هذا اللون على:
   - خلفية الـ Header
   - لون زر التشغيل
   - لون شريط التقدم في Now Playing

```dart
final palette = await PaletteGenerator.fromImageProvider(
  FileImage(File(album.coverPath!)),
);
final dominantColor = palette.dominantColor?.color ?? Colors.purple;
```
