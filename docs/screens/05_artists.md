# 5️⃣ Artists Screen

## الوصف

تعرض قائمة بجميع الفنانين الموجودين في مكتبة الموسيقى مع عدد أغانيهم.

---

## التخطيط — قائمة الفنانين

```
┌─────────────────────────────────┐
│  Artists (32)      [🔍] [Grid▾] │
├─────────────────────────────────┤
│  A                              │  ← حرف أبجدي فاصل
│  ┌───────────────────────────┐  │
│  │ 👤 │ Artist Name          │  │
│  │    │ 12 أغنية · 3 ألبومات│  │
│  └───────────────────────────┘  │
│  ┌───────────────────────────┐  │
│  │ 👤 │ Another Artist       │  │
│  │    │ 5 أغاني              │  │
│  └───────────────────────────┘  │
│  B                              │
│  ...                            │
├─────────────────────────────────┤
│  ▶ Mini Player                  │
└─────────────────────────────────┘
```

---

## طرق العرض

| الطريقة | الوصف |
|---------|-------|
| List View | الافتراضي — عرض قائمة مع تفاصيل |
| Grid View | شبكة مع صورة الفنان (إن وُجدت) |

## البيانات

```dart
class Artist {
  final int id;
  final String name;
  final int numberOfTracks;
  final int numberOfAlbums;
  final String? coverPath;      // صورة مستخرجة من إحدى الأغاني
}
```

## ملاحظات

- صورة الفنان تُستخرج من أول ألبوم له (Artwork)
- `on_audio_query` يوفر `queryArtists()` مباشرة
- الفرز الأبجدي مع `StickyHeaders` للحروف
