# 9️⃣ Now Playing Screen ⭐ (أهم شاشة)

## الوصف

شاشة التشغيل الرئيسية. تظهر عند النقر على أغنية أو على الـ Mini Player. أهم شاشة من حيث التصميم والتفاعل.

---

```

│   ┌─────────────────────────┐  │
```

## عناصر الشاشة التفصيلية

### صورة الألبوم
- حجم كبير (300×300 أو أكبر)
- تأثير دوران عند التشغيل (Vinyl Record effect)
- يتوقف الدوران عند الإيقاف
- إذا لم توجد صورة → رمز موسيقى افتراضي مع لون Accent
- Swipe يسار/يمين للأغنية التالية/السابقة

### خلفية الشاشة
- **Dynamic Color**: استخراج لون سائد من صورة الألبوم
- تطبيق gradient من هذا اللون في خلفية الشاشة
- الانتقال بين الألوان بـ animation سلس

### شريط التقدم
- `Slider` قابل للسحب
- عرض الوقت الحالي (يسار) والوقت الكلي (يمين)
- تغيير موضع التشغيل بالنقر أو السحب

## أزرار التحكم

| الزر | الوظيفة | الحالات |
|------|---------|---------|
| ⏮ | السابقة / البداية | - |
| ▶/⏸ | تشغيل / إيقاف مؤقت | كبير، في المنتصف |
| ⏭ | التالية | - |
| 🔀 | Shuffle | On / Off |
| 🔁 | Repeat | Off / One / All |
| ❤️ | مفضلة | مفعّل / غير مفعّل |

## Equalizer

يفتح Equalizer النظام المدمج في Android:

```dart
_openSystemEqualizer(int audioSessionId) async {
  try {
    await AndroidContentResolver.openEqualizer(audioSessionId);
  } catch (e) {
    // عرض Equalizer مخصص بسيط
    showCustomEqualizer();
  }
}
```

أو عرض Equalizer مخصص بـ 5 bands:
```
60Hz  |  230Hz  |  910Hz  |  4kHz  |  14kHz
 ↑        ↑        ↑        ↑        ↑
[-12 .. 0 .. +12 dB]
```

## إيماءات الشاشة (Gestures)

| الإيماءة | الوظيفة |
|----------|---------|
| Swipe يسار | الأغنية التالية |
| Swipe يمين | الأغنية السابقة |
| Swipe للأسفل | إغلاق الشاشة (العودة للـ Mini Player) |
| ضغطة مزدوجة على الصورة | إضافة / إزالة من المفضلة |
