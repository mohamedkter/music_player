# 🎵 نظرة عامة على المشروع

## الفكرة

تطبيق **مشغل موسيقى محلي** يعمل على Android، يتيح للمستخدم تصفح وتشغيل ملفات الصوت المحفوظة على الجهاز بواجهة أنيقة واحترافية.

---

## 🎯 الهدف

بناء أفضل مشغل موسيقى محلي من حيث:
- **السرعة** في تحميل المكتبة الموسيقية
- **الواجهة** الجميلة مع دعم Dynamic Color
- **الميزات** الاحترافية (Equalizer, Sleep Timer, Queue...)
- **التصفح المرن** حسب الفنان / الألبوم / المجلد / Playlist
